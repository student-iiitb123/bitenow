Food Ordering & Restaurant Dashboard (Next.js)


A full-stack  food ordering application built with **Next.js**, **MongoDB**, and **Tailwind CSS**.  
The platform allows users to browse restaurants, view menus, place orders, while restaurants manage food items through a dedicated dashboard.


Project Overview

This project replicates core functionalities of Zomato, focusing on real-world application architecture, REST APIs, dynamic routing, and database-driven UI updates.

It is designed to demonstrate **full-stack development skills**, including frontend, backend, database design, and API handling.


 User Features
- Browse restaurants
- View food menus
- Add items to cart
- Place food orders
- Responsive UI


 Restaurant Dashboard
- Restaurant authentication
- Add food items
- Edit food items
- Delete food items
- View restaurant-specific menu
- Manage orders


 Tech Stack

### Frontend
- Next.js (App Router)
- React
- Tailwind CSS

 ### Backend
- Next.js API Routes
- RESTful APIs

### Database
- MongoDB
- Mongoose


Project Architecture

- Uses **dynamic routing** for food and restaurant pages
- Follows **controlled components** for form handling
- API routes handle CRUD operations
- MongoDB stores users, restaurants, food items, and orders
- Clean separation of frontend and backend logic
